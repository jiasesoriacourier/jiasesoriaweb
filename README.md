# J.I Asesoría & Courier - Sitio corporativo

Proyecto base en Next.js para la nueva página corporativa.

## Cómo usar
1. Instala dependencias:
   npm install

2. Ejecuta en desarrollo:
   npm run dev

3. Abre:
   http://localhost:3000

## Deploy
Este proyecto puede subirse a GitHub y desplegarse en Render o Vercel.
